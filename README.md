# Проект по компьютерному зрению: Отслеживание футболистов на поле

**Описание**  
Этот проект реализует систему для обнаружения и отслеживания футболистов на футбольном поле с использованием технологий компьютерного зрения. Проект состоит из двух основных частей: обучения модели и использования модели для отслеживания игроков на поле.

## Структура проекта

Проект состоит из двух файлов:

- `90ep_model_training.ipynb` — файл для обучения модели.
- `Player tracking model.ipynb` — файл для использования обученной модели для отслеживания футболистов.

Все библиотеки и зависимости которые необходимо установить указаны в файлах.
